vector< string > KnownComms [static, private]

This is a vector their string labels.

At the outset two communicators are known: MPI_COMM_WORLD and MPI_COMM_SELF (both all caps). The user may add more comms by calling NewComm with some string identifier for the new comm, these identifiers will be case sensitive.
