Detailed Description

The top level class in charge of holding molecular details.

The molecule_class object is basically a set of atoms (just like a real molecule). It knows the basic details of the molecule, but nothing too fancy. One key concept that is used to decrease the constructor argument list is the idea of "messy" coordinates. This is simply an Natoms by 4 array where element i,0 is the atomic number of atom i, and i,1 to i,3 are the x,y, and z coordinates of atom i, respectively.

void WriteGTFockXYZFile (  )  const

This function writes the molecule as an *.xyz file to the current directory.

In order to interface with the GTFock library we need an *.xyz file. This routine will make that file. Note that for whatever reason, the GTFock library does not include the standard comment line in the *.xyz file, so we also do not.
