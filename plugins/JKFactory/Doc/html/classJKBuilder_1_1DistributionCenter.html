Detailed Description

A distribution center is a JKFactory that makes distributed objects.

The name distribution center is supposed to be a pun related to the name "factory". It is a lame one, but I like it, so it stays. Either way, this is an abstract base class that actual Distributed routines derive from. For example the GTFock library is a derived class of this class.
