Sets up a bare-bones, distributed, JK object in the DesiredBasis, on the desired molecule.

The JK object minimally needs three things: a basis, a molecule, and density matrix. The Carts array should be in "messy" form, which is:
<atomic number>=""> <x-coordinate> <y-coordinate> <z-coordinate>
this way we do not have to pass the identities in separately. Also note the atomic numbers will be mapped to integers via an int cast. To be on the safe side you may want to use numbers slightly bigger then the desired value, e.g. 6.01 for carbon; however simply setting it to 6.000000000000....should be OK as well. Also note that JK objects can only be used for one molecule and basis set at a time.

SharedMatrix T [private]

The transformation matrix to ERD's ordering.

For Cartesian Gaussians ERD is ordered (lx,ly,lz): 1. L 0 0 2. L-1 1 0 3. L-1 0 1 4. L-2 2 0 5. L-2 1 1 6. L-2 0 2 7. L-3 3 0 ....

In general the loop looks like: for (int i=0;i<L;i++){ for(int j=0;j<i;j++){ Lx=L-i; Ly=i-j; Lz=j; } }

This is the same for Psi4

For spherical Gaussians Psi's order is 0,1,-1,...,L,-L
