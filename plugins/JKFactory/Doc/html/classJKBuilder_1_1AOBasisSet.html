Detailed Description

The top-level basis set object for the JKBuilder library.

ERD and OED integrals are run by shell, so we need details per shell; however each of those shells retains some properties of the atoms they reside on. I have made the decision to store the shells for each atom together as objects. The set of all shells for an atom is a basis set for that atom, an JKBuilder::AtomicBasisSet object. The set of these basis sets for each atom in a molecular system is then the AOBasisSet. That is what is stored in this object.

std::vector<AtomicBasisSet*> AOBasis [private]

The AO basis.

A vector of the basis sets for each atom, this is the AO basis. The order of the AtomicBasisSets that are on it is expected to correspond to the order the atoms are in the input file. That is AOBasis[i] should be the basis set of atom i,counting from 0.
