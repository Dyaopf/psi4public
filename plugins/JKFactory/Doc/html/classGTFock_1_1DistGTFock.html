The function in charge of establishing our link to the GTFock library.

The GTFock library needs several things from us to get started: it's basis set object, the number of processors per row and column of the Fock matrix, and some basic info about the problem. This routine takes care of that.

The function actually in charge of building the J and K matrices.

This function first updates the density matrix to be the one given to it. Next, it computes J and K and returns them as matrix objects.
