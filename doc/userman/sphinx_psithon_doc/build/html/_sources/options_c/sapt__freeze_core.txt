
.. glossary::

   FREEZE_CORE
      Sets the number of core orbitals to freeze in the evaluation
      of the :math:`E_{disp}^{(20)}` and :math:`E_{exch-disp}^{(20)}`
      terms. It is recommended to freeze core in all SAPT computations.

      * **Type**: string
      * **Possible Values**: TRUE, FALSE, SMALL, LARGE
      * **Default**: FALSE
