Gallery¶

The gallery lists examples of all plots and available renderers.

Labeled data¶

These Renderers require DataTypes.LabeledData. Labeled data is consists of a list of labels and a list of data, for example [ ("column1", "column2" ), ( 10, 20) ].

Table¶

The Renderer.RendererTable class inserts labeled data into a table.

Stacked Barplot¶

The Renderer.RendererStackedBars class presents labeled data as stacked bars.

Interleaved Barplot¶

The Renderer.RendererInterleavedBars class presents labeled data as interleaved bars. Both interleaved-bars and bars can be used.

Single column data¶

Renderers that accept a single column of data of type DataTypes.SingleColumnData per track and slice, for example [2,3,1,3,4].

Histogram¶

The Renderer.RendererHistogram class computes a histogram of data and inserts it as a table.

HistogramPlot¶

The Renderer.RendererHistogramPlot class computes a histogram of data and inserts a plot.

Stats¶

The Renderer.RendererStats class computes summary statistics and displays them in a table.

Boxplot¶

The Renderer.RendererBoxplot class computes boxplots.

Multi-column data¶

Renderers that accept multiple columns of data of type DataTypes.SingleColumnData per track and slice. The data is asscociated with column headers, for example [ ('column1', 'column2'), ( ( 1,2,3), (4,5,6) )].

Pairwise statistics table¶

Compute correlation statistics between all columns.

Pairwise statistics plot¶

Plot correlation coefficients between all columns.

Pairwise scatter plot¶

A scatter plot.

Grouped table¶

A grouped table.
