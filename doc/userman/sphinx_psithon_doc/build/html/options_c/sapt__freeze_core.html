FREEZE_CORE

Sets the number of core orbitals to freeze in the evaluation of the and terms. It is recommended to freeze core in all SAPT computations.

  • Type: string
  • Possible Values: TRUE, FALSE, SMALL, LARGE
  • Default: FALSE
