
.. glossary::

   NO_RESPONSE
      Don't solve the CPHF equations, evaluate :math:`E_{ind}^{(20)}`
      and :math:`E_{exch-ind}^{(20)}` instead of their response-including
      counterparts. Only turn on this option if you are not going to 
      use the induction energy.
    
      * **Type**: `boolean`_
      * **Default**: FALSE
